# AI Research Paper Analyzer

An AI-powered tool that helps students and researchers deeply analyze
research papers — beyond simple summarization.

## Features
- Smart section-wise summarization
- Research gap detection
- Future work suggestions
- Q&A on any paper (RAG-based)
- Citation extractor
- Related paper recommendations

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** FastAPI, Python
- **AI:** Gemini API
- **Vector DB:** ChromaDB
- **PDF Processing:** PyMuPDF, pdfplumber
